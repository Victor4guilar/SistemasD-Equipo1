# CHANGELOG

## Código C

### 1.0.0 – 2026-03-02 – Axel

Se crea la primera versión de la calculadora básica. Realiza una operación fija, aunque contenía un error (realizaba una resta en lugar de suma).

### 1.0.1 – 2026-03-24 – Jonathan

Se corrige el error en la operación (de resta a suma).

### 1.1.0 – 2026-03-25 – Gabriel

Se agrega entrada de datos por teclado.
Se implementa validación de datos con manejo de errores para evitar entradas no numéricas.

### 1.2.0 – 2026-03-25 – Víctor

Se optimiza el código eliminando variables innecesarias.
Se mantiene la validación de datos con manejo de errores.

### 2.0.0 – 2026-03-25 – Jonathan

Se transforma la calculadora en una versión completa.
Ahora permite realizar suma, resta, multiplicación y división.
Se añade menú interactivo y validación de entradas, incluyendo control de división entre cero.
